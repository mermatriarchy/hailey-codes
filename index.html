About

Hailey Miller is currently a student in the Web Development Immersive at General Assembly in New York. He's most interested in full-stack web development with an emphasis on Ruby on Rails and JavaScript. His work focuses on using progressive enhancement to create elegant, responsive web design for optimal user-experience.

More Resume

Hailey first started working in the tech industry as a translator and content creator for a PR firm representing start-ups. After being hired at an eCommerce company as a project manager, he started to teach himself the basics of coding to improve his knowledge-base for scoping projects. As it turns out, Hailey thought coding was fun, and decided to make the switch to programming.
