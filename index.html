About

Hi there, I'm a Web Developer working at the fintech company Laurel Road. I've also worked as a freelance developer for Penguin, and as a part-time Instructional Associate for the Front-End Web Development course at General Assembly. I'm most interested in full-stack web development, particularly Ruby on Rails, JavaScript and Test-Driven Development. My work focuses on using progressive enhancement to create elegant, responsive web design for optimal user-experience.

More Resume

I first started working in the tech industry in 2013 as a translator and content creator for a PR firm representing start-ups. After being hired at an eCommerce company as a project manager, I started to teach myself the basics of coding to improve my knowledge-base for scoping projects. As it turns out, I thought coding was fun, and decided to make the switch to programming. I graduated from the Web Development Immersive at General Assembly in mid-2016. Outside of work, I've been solidifying my React skills and learning fundamental computer science concepts through various online resources.
